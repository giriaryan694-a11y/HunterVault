## HOW TO USE

### 1. OPENING THE APPLICATION
   - Simply open the HTML file in any modern web browser
   - No installation or server required
   - Works offline after initial load

### 2. NAVIGATION
   - Use the search bar to find specific tools
   - Scroll through categorized sections
   - Click theme toggle to switch between Dark/Light mode
   - Hover over tool cards for elevation effect

### 3. SEARCHING FOR TOOLS
   - Type in the search box at the top
   - Results appear in real-time
   - Categories with matching tools remain visible
   - Clear search box to show all tools again

### 4. THEME MANAGEMENT
   - Click the theme button (top-right)
   - Choice is automatically saved
   - Will remember your preference on next visit
